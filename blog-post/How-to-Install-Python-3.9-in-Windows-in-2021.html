How to Install Python 3.9 in Windows in 2021

In this blog post I am going to talk about the stps to Install Python Latest Version In Windows Mac and Linux

Before installing Python, check if Python is already installed on your computer. Steps to check if Python is already installed or not:

  1. Open Windows Powershell.
  2. Type "python --version" and press enter

 

If an error message shown in the above gif is displayed, Python is not installed on your computer. There is a chance that an older version of Python is already installed on your computer. In this case, you need to uninstall the older version and install the latest version.

Follow the steps given below to install the latest version of Python in Windows:

Step 1: Click here and download the latest version of Python available. At the time of writing this article, the latest version is Python 3.9.1.

Step 2: Install the .exe file as soon as the downloading process completes. All you need to do is to keep pressing next until python installs!
